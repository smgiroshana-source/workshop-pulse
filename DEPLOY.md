# Workshop Pulse v2.1.0 - Deployment Guide

## Quick Deploy (Vercel)
1. Push to GitHub
2. Import in Vercel
3. Deploy

## Local Development
```bash
cd workshop-pulse
npm install
npm run dev
```
Open http://localhost:3000

## Features (v2)
- 3 work types: Paint & Body, Mechanical, Both
- 3 job types: Insurance (12-stage), Direct Major, Quick Job
- Cost tracking: Ex-stock, Purchased, Outsource, Labour
- Customer registry with vehicle reg + phone normalization
- Parts quotation with approval workflow
- Invoice generation with dual discounts
- QC checklist + follow-up system
- PDF generation for estimates, invoices, PQ
- WhatsApp sharing
- Tablet + mobile responsive
